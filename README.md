📖 Overview
This repository contains a collection of C++ source code developed as part of the OOP (CSE 3011) curriculum. The projects focus on mastering core programming paradigms, ranging from basic syntax to advanced architectural patterns.

🛠 Tech Stack
Language: C++11/14/17

Compiler: GCC / Clang / MSVC

Development Environment: VS Code / Visual Studio

📂 Project Structure
The repository is organized into modules representing different stages of the learning journey:

Plaintext
OOP-VJU/
├── OOP1/           # Fundamentals: Classes, Objects, and Constructors
├── OOP2/           # Relationship: Inheritance and Composition
├── OOP3/           # Abstraction: Polymorphism and Virtual Functions
├── OOP4/           # Advanced: Templates, STL, and Exception Handling
├── .gitignore      # Build artifacts and temporary files
└── README.md       # Project documentation
🚀 Key Concepts Covered
Encapsulation: Protecting data by using access specifiers (public, private, protected).

Inheritance: Reusing code through hierarchical class structures.

Polymorphism: Utilizing virtual functions and method overriding for dynamic behavior.

Abstraction: Creating interfaces and abstract base classes.

Memory Management: Understanding pointers, references, and the Rule of Three/Five.

🔧 Building and Running
To compile and run the programs, ensure you have a C++ compiler installed (e.g., g++).

Clone the repository:

Bash
git clone https://github.com/DevK4z/OOP-VJU.git
Navigate to a specific module:

Bash
cd OOP-VJU/OOP1
Compile the source file:

Bash
g++ main.cpp -o main
Execute the binary:

Bash
./main
